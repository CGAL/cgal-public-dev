\begin {ccRefClass} {Lines_through_segments_traits_3<Rational_kernel, Alg_kernel>}
    
\ccDefinition 

The class \ccStyle{Lines_through_segments_traits_3<Rational_kernel, Alg_kernel>} is a model of the
\ccc{LineThroughSegments_3} concept, currently it is the only model supplied with the package.
This class must be instantiated with 2 exact geometric kernels. The first is a rational kernel, which is used for the input line segments, and the second is an algebraic kernel, which is used for the output objects.
