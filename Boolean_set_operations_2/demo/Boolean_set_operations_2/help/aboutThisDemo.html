This program demonstrates several operations on general polygons offered by the 2D Regularized Boolean Set Operations and 2D Minkowski Sums packages of CGAL.

• You can create, manipulate, save, and restore general polygons with holes bounded by three types of curves, namely, line segments, line segments or circular arcs, and arcs of Bezier curves. Minkowski sum can be applied only to polygons of the former type.
• The program maintains buckets of polygons. Each bucket contains a set of interior-disjoint general polygons with holes. The polygons of a bucket are rendered in a distinct color associated with the bucket; hence called a color bucket. The program starts with three empty color buckets, but you can add more buckets up to a maximum of seven.
• Operations are applied to entire buckets, that is, an operand is the set of points that comrise all polygons of a given bucket. The result of an operation is inserted into an empty bucket.
• You can apply unary operations, i.e., complement, move, and copy, binary operations, i.e., difference, symetric difference, and Minkowski sum, or operations that accept at least two point sets (color buckets), i.e., intersection and union.
• You can save your active point set (bucket) in a file or restore previously saved point sets.
• Each operation, including render, save, or restore, is considered a single step.
• If you feel that you made a mistake, you can always go back as many steps as you want using Undo operation.
• You can drag the scene using the Pan operation and also Recenter the viewport.
• Finally, you can Clear selected buckets or Reset the entire program to enter the starting state.
