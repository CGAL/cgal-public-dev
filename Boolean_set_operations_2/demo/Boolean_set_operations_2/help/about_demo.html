Zoom out

Using mouse :Scroll up the using scroll button of the mouse.
Using touchpad:Move both the fingers in same direction at equal speed on touchpad.

 

 

Zoom in

Using mouse :Scroll down the using scroll button of the mouse.
Using touchpad:Move both the fingers in opposite direction at equal speed on touchpad..

 

 

PAN Application

Press Ctrl+P combination keys or go to View menu and choose "PAN application". Initially, the pan operation will not work because, the polygons are tightly fit in the screen. The user needs to first zoom out and than zoom in again to perform PAN operation in the newly created area.

 

 

Recentre the screen

Press Ctrl+R combination keys or go to View menu and choose "Recenter Scene". When you recenter scene the application, all the polygons, the screen will resize to just fit all the polygons on the screen.

 

 

Quit the application

Press Ctrl+Q combination keys or go to File menu and choose "Quit". When you quit the application, all the windows you have created will be destroyed.

 

 

Perform a boolean operation

Left click on one of the boolean operations in Operations menubutton layer (Intersection, Union, Difference between any two polygon, Symmetric Difference, Complement of any polygon). the result polygon will be green.
Shortcuts
Complement Ctrl+c
Intersection Ctrl+i
Union Ctrl+u
Symmetric Difference Ctrl+s
Difference Ctrl+d

 

 

Set the current active polygons

There are 7 types of polygons Blue, Red, Black, Brown, Yellow, Magneta and Aqua. To draw polygon of any type toggle the radio button the respective polygon. All inserted polygons will be of the selected color type.

 

 

Insert circular polygon

Left click on the "Insert Circular Polygon" in Operations menu layer. The user is not allowed to have linear polygon with circular polygons on the screen at the same time. So, if there is/are already some linear polygons in the screen, a warning will pop up, which will ask you to either clear linear polygons or continue with linear polygons. Click "Yes" to draw circular polygons.
A new vertex is inserted every time the left mouse button is pressed, if the polyline entered so far is simple. To insert a circular arc press the left mouse button while pressing the Ctrl key, and the cursor will be moved to the middle point of the current segment and moving the mouse will change the circular arc, another left click will add the circular arc if it doesn't intersect any other edge. Holding left mouse key will also perfom the same operation as like pressing the Ctrl key.
A right click closes the polygon, if it is simple.
You can use the Escape key if you want to remove your last entered point in the polygon. The new polygon is created.

Left click on the "Insert linear Polygon" in Operations menu layer. The user is not allowed to have circular polygon with linear polygons on the screen at the same time. So, if there is/are already some circular polygons in the screen, a warning will pop up, which will ask you to either clear circular polygons or continue with circular polygons. Click "Yes" to draw linear polygons.
A new vertex is inserted every time the left mouse button is pressed, another left click will add the linear curve if it doesn't intersect any other edge.
A right click closes the polygon, if it is simple.
You can use the Escape key if you want to remove your last entered point in the polygon. The new polygon is created.

 

 

Delete current polygons

Left click on the "Delete current polygon" in Edit menu , all polygons of the selected type will be deleted.

 

 

Delete all polygons

Left click on the "Delete all polygons" in Edit menu , all polygons will be deleted.
